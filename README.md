## 프로젝트 실행 방법
1. 프로젝트 최상단 폴더에 .env 폴더를 생성합니다. 생성 후 다음 KEY를 입력합니다.
  ```
  REACT_APP_API_URL=https://pre-onboarding-selection-task.shop
  ```
  
2. 프로젝트 실행을 위해 패키지를 설치 합니다.
  ```
  $ npm install
  ```
  
3. 프로젝트를 실행 합니다.
  ```
  $ npm start
  ```
